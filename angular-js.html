Angular JS - Pengembangan Web Tingkat Lanjut

AngularJS adalah framework software open-source yang digunakan untuk membuat aplikasi single-page berbasis website. Framework ini juga cukup populer bagi para developer untuk membuat menu animasi di laman web HTML.

Sejarah Angular JS sendiri dulunya adalah gagasan dari seorang engineer Google, Misko Hevery dan Adam Abrons. Google merilis versi pertamanya pada tahun 2012. Sejak saat itu, AngularJS terus berkembang.

Sebenarnya, sebelum era framework ini, ada banyak cara untuk membuat laman web dinamis. Namun, metode tersebut tidak semudah dan senyaman framework Angular JS.

AngularJS menggunakan arsitektur Model-View-Controller (MVC) yang digunakan dalam pengembangan aplikasi berbasis web.
